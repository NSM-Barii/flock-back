Fixing BLE Scanning on Kali Linux (BlueZ + Bleak)

1. Install BlueZ packages:
    sudo apt update
    sudo apt install bluez bluez-tools bluez-firmware

2. Enable & start Bluetooth service:
    sudo systemctl enable bluetooth
    sudo systemctl start bluetooth

3. Check Bluetooth service status:
    systemctl status bluetooth

4. Restart DBus:
    sudo systemctl restart dbus

5. Unblock and enable Bluetooth adapter:
    sudo rfkill unblock bluetooth
    sudo hciconfig hci0 up

6. Test Bluetooth manually:
    bluetoothctl
    power on
    scan on

7. Run your BLE scanner:
    python3 ble_scan.py

If anything fails, run:
    systemctl status bluetooth
    rfkill list
    hciconfig -a
    lsusb
