# GPS Setup Instructions for Debian/Kali Linux

1. Install dependencies:
   sudo apt update
   sudo apt install gpsd gpsd-clients python3-gps python3-pip
   pip3 install gps3

2. Create fixed GPS device name (optional but recommended):
   sudo nano /etc/udev/rules.d/99-gps.rules
   # Add this line (replace XXXX and YYYY with actual ID from `lsusb`)
   SUBSYSTEM=="tty", ATTRS{idVendor}=="XXXX", ATTRS{idProduct}=="YYYY", SYMLINK+="gps0"
   sudo udevadm control --reload

3. Configure GPSD:
   sudo nano /etc/default/gpsd
   # Set these values:
   START_DAEMON="true"
   GPSD_OPTIONS="-n"
   DEVICES="/dev/gps0"
   USBAUTO="true"

4. Enable and start GPSD:
   sudo systemctl enable gpsd
   sudo systemctl start gpsd

Now gpsd will autostart and connect to the GPS on boot.
